# 页面文字配置说明

## 这些文字都在哪里配置？

所有页面上的文字都是通过**国际化（i18n）配置文件**控制的，不是写死在模板中的。

### 1. **Recent Posts（最新文章）**
- **模板位置**：`themes/mainroad/layouts/partials/widgets/recent.html`
- **使用的翻译键**：`{{ T "recent_title" }}`
- **配置文件**：`/i18n/zh-cn.yaml`
  ```yaml
  - id: recent_title
    translation: 最新文章
  ```

### 2. **Categories（分类）**
- **模板位置**：`themes/mainroad/layouts/partials/widgets/categories.html`
- **使用的翻译键**：`{{ T "categories_title" }}`
- **配置文件**：`/i18n/zh-cn.yaml`
  ```yaml
  - id: categories_title
    translation: 分类
  ```

### 3. **Tags（标签）**
- **模板位置**：`themes/mainroad/layouts/partials/widgets/taglist.html`
- **使用的翻译键**：`{{ T "tags_title" }}`
- **配置文件**：`/i18n/zh-cn.yaml`
  ```yaml
  - id: tags_title
    translation: 标签
  ```

### 4. **主页**
- **模板位置**：`themes/mainroad/layouts/404.html`（404页面中的链接）
- **使用的翻译键**：`{{ T "404_linktext" }}`
- **配置文件**：`/i18n/zh-cn.yaml`
  ```yaml
  - id: 404_linktext
    translation: "主页"
  ```

## 如何修改这些文字？

### 方法 1：直接编辑配置文件（推荐）

编辑项目根目录的 `/i18n/zh-cn.yaml` 文件：

```yaml
# Recent Posts widget
- id: recent_title
  translation: 最新文章  # 改成你想要的内容

# Categories widget
- id: categories_title
  translation: 分类  # 改成你想要的内容

# Tags List widget
- id: tags_title
  translation: 标签  # 改成你想要的内容
```

### 方法 2：创建项目级配置文件覆盖主题配置

如果你不想修改主题文件，可以在项目根目录创建 `/i18n/zh-cn.yaml`，Hugo 会优先使用项目级的配置。

## 为什么页面还是显示英文？

如果页面仍然显示英文（如 "Recent Posts" 而不是 "最新文章"），可能是以下原因：

### 1. Hugo 服务器没有重启
```bash
# 停止当前服务器（Ctrl+C），然后重新启动
hugo server -D
```

### 2. 配置文件路径不对
确保文件位于：
- `/i18n/zh-cn.yaml`（项目根目录）
- 或 `/themes/mainroad/i18n/zh-cn.yaml`（主题目录）

### 3. 语言代码不匹配
检查 `config.toml` 中的 `languageCode` 是否为 `zh-cn`：
```toml
languageCode = "zh-cn"
```

### 4. YAML 格式错误
检查 YAML 文件语法是否正确，确保缩进使用空格而不是 Tab。

### 5. 浏览器缓存
按 `Ctrl+F5`（Windows）或 `Cmd+Shift+R`（Mac）强制刷新页面。

## 所有可配置的文字列表

以下是所有可以通过国际化配置的文字：

| 页面位置 | 翻译键 | 中文翻译 | 英文原文 |
|---------|--------|---------|---------|
| 侧边栏 | `recent_title` | 最新文章 | Recent Posts |
| 侧边栏 | `categories_title` | 分类 | Categories |
| 侧边栏 | `tags_title` | 标签 | Tags |
| 侧边栏 | `search_placeholder` | 搜索... | SEARCH... |
| 菜单 | `menu_label` | 菜单 | Menu |
| 文章导航 | `post_nav_prev` | 上一篇 | Previous |
| 文章导航 | `post_nav_next` | 下一篇 | Next |
| 404页面 | `404_title` | 404. 页面未找到 | 404. Page not found |
| 404页面 | `404_linktext` | 主页 | Home |
| 目录 | `toc_title` | 页面内容 | Page content |

完整的翻译列表请查看 `/i18n/zh-cn.yaml` 文件。

## 验证配置是否生效

1. **检查 HTML 源码**：
   - 打开页面，查看源码
   - `<html>` 标签应该是 `<html lang="zh-cn">`

2. **查看侧边栏**：
   - 应该显示"最新文章"而不是"Recent Posts"
   - 应该显示"分类"而不是"Categories"
   - 应该显示"标签"而不是"Tags"

3. **测试 404 页面**：
   - 访问一个不存在的页面（如 `/test-404`）
   - 应该显示中文错误信息
