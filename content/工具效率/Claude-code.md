# claude code 替代品

1. 安装 claude

```bash
npm install -g @anthropic-ai/claude-code
claude --version # 查看版本
```

2. 安装 CC switch (https://ccswitch.io/zh/)

3. 获取 agnes api-key 有免费版本 https://platform.agnes-ai.com/settings/apiKeys

4. 在 CC Switch 新增 agnes 模型填写

- api-key:(**\*\*\*\***),
- 请求地址(https://apihub.agnes-ai.com/v1)

- 高级选项 -> api 格式选择需开启路由
- 模型映射，点击获取模型列表，将获取的模型列表在最右边下拉选择，做好选择agnes-2.0-flash 免费版本。点击支持 1M
- 保存配置后，需要 设置路由开关，在 CC Switch 软件的左上方锯齿按钮设置，选择路由页签，点击本地路由，将路开开关打开，路由启用选择Claude.

5. 在命令窗口就可以使用claude了

6. 在 Trae 中安装 claude code 插件就可以 使用 claude code Ai助手编程了
