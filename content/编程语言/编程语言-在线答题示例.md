{{< quiz id="quiz-2" question="`try-catch-finally` 中执行 `System.exit(0)`，`finally` 会执行吗？" correct="A" >}}
{{< quizoption value="A" explanation="正确：`System.exit(0)` 会直接终止 JVM 进程，所有后续代码（包括 finally 块）都不会执行。这是唯一能让 finally 不执行的情况。" >}}
不会，`System.exit(0)` 直接终止 JVM，`finally` 被跳过。
{{< /quizoption >}}
{{< quizoption value="B" explanation="错误：finally 块的执行与 try 是否抛异常无关，只要 JVM 正常运行，finally 总是会执行。" >}}
只有 `try` 未抛异常才执行。
{{< /quizoption >}}
{{< quizoption value="C" explanation="错误：虽然一般情况下 finally 会执行，但如果调用了 System.exit()，finally 就不会执行。" >}}
会，`finally` 总会执行。
{{< /quizoption >}}
{{< quizoption value="D" explanation="错误：finally 的执行与 catch 捕获的异常类型无关，只要 JVM 正常运行就会执行。" >}}
取决于 `catch` 是否捕获 `RuntimeException`。
{{< /quizoption >}}
{{< /quiz >}}
