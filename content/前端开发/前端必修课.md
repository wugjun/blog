# 代码执行结果

```javascript
const obj = {
  a: 0,
};
obj["1"] = 0; //所有对象属性类型都会转化为 字符串,所有数字类型的属性都要置前升序排序 ,obj: { 1:0 ,a:0}
obj[++obj.a] = obj.a++; //obj: { 1:0 ,a:2}

const values = Object.values(obj); //获取对象所有属性值，返回一个数组 [1,2]
obj[values[1]] = obj.a; //将数组第二个元素赋值为 obj.a  obj[1] = 2
console.log(obj); //{"1": 1, "2": 2,"a": 2}
```

# 为什么需要虚拟dom

1. 性能优化：虚拟dom可以减少真实dom的操作次数，提高渲染性能。
2. 跨平台：虚拟dom可以在不同的平台上运行，例如浏览器、移动端等。
3. 组件化：虚拟dom可以将页面拆分成多个组件，每个组件都有自己的状态和行为，方便开发和维护。
